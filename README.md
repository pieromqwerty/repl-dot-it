repldotit.com
=======

An online environment for interactively exploring programming languages, based
on [Repl.it](https://github.com/replit-archive/repl.it).

Current Languages
-----------------

* JavaScript Variants
  * JavaScript
  * CoffeeScript
  * Kaffeine
  * Move
  * JavaScript.next

* Esoteric
  * Bloop
  * Brainfuck
  * LOLCODE
  * Unlambda
  * Emoticon

* Classic
  * Quick Basic
  * Forth

* Serious
  * Scheme
  * Lua
  * Python
  * Ruby (beta)
